# Route
Networking utilities made easy with ***Route Language***. Download the interpreter from the releases tab now!

## Create a program
```route
# Hello world -- Route
OUTPUT "Hello, World"
```

## Run
```bash
route.exe ./hello.route
```
***Note that the file should end in .route extension***

Run in the interactive mode:
```bash
route.exe
route> OUTPUT "Hello, world!"
```
